<p><strong>The UV-LED Light Source Market Analysis by types is segmented into:</strong></p>
<p><ul><li>Linear Irradiation</li><li>Spot Irradiation</li><li>Surface Irradiation</li></ul></p>
<p><p>The UV-LED light source market is segmented into three main types: linear irradiation, spot irradiation, and surface irradiation. Linear irradiation utilizes elongated sources to evenly cover long, narrow areas, making it ideal for applications like curing coatings in industrial processes. Spot irradiation focuses UV light on specific, targeted areas, suitable for precision applications like printed circuit board curing. Surface irradiation provides uniform UV exposure across larger surfaces, effectively used in sanitization and film curing. Each type caters to distinct application needs.</p></p>
